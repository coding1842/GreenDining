# 프로젝트 소개  
- __GreenDining__ 프로젝트 

- 식자재를 1차 생산자가 직접 등록 및 판매하도록 하여 중간 유통과정을 생략하여 생산자들의 소득을 증가시키고 신선한 식자재를 소비자가 보다 저렴하게 소비할 수 있습니다.  
- 계절별로 신선한 제철음식을 구입할 수 있습니다.
